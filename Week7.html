Week 7

For this weeks project we have been given the task to print a ring of our choice. This will be done by using SLA printing. SLA printing is a printing process different from traditional 3D printing methods as it uses a unique material specific to this methid. SLA Printers use a Plastic Material called thermoset liquad as apposed to a thermoplastic in 3D printing. Through this process this liquid is set and UV-Cured in layers in order to form the final product.

Beginning Comments

Before following there are a couple comments that need to be made. It is important to note that for this project we were given multiple tutorial videos associated with the ring of our choice. That being said these videos have their own pros and cons. For example, the video related to our ring (the spinning ring) has numerous problematic cons that can easily confuse and or frustrate someone trying to replicate this process. Some cons found in this video are as simple as not having the text in english. This may be a little redundant however the person teaching the tutorial is speaking in english. Another con, the measurements and units are european units, again, this can easily make the process just a little bit harder for someone not used to using these units. Finally, the speed of which the teacher teaches as is way to fast for such a complicated process. I had found myself sometimes watching frame by frame and or listening to it at half speed just to follow him. That being said, the person teaching clearly knows what they are doing and shows us step by step what needs to be done to the exact setting and unit which I can respect and made the process less stressful. At the end of the video he specified what the final product should look like and also gave us some special tricks to help us see what it could look like using the render tool.

1. Beginning Sketch

Just like any assignment that involved 3D Printing we need to boot up fusion360 and create a new sketch from there we need to start following the provided tutorial according to your chosen ring. In our case we are desgining a spinning ring.

2. Following the tutorial

Following the tutorial we begin by using our first new tool, the line segment tool. This tool will help us sketch out the beginning pieces for the inside and outside pieces of our ring. Because the video is using metirc units make sure to follow the units he is using seeing as if you dont your ring will be extremely off. Proceed by follow these steps to complete this sketch.

Make sure that your sketch looks like this with the measurements as shown. Finish your sketch and move on to the next step.

3. Revolve

Here we are going to be using our second tool, Revolve Tool. Your final product after finishing your sketch should look like this. Make sure that these are two selectable objects.

Select both objects and go to create and then revolve.

After selecting both of your objects make sure to input 360 degrees. From there you should be done with this step.

Here is what your product should look like once you are done.

4. Inner Mechanics

Here is our next set of tools that we are going to be using for this process. Go to the inspect tab where the circled drop down menu is. From there select "Selective Analysis". Here is what it should look like.

5. Fillet Edges

We are going to do two sets of fillets. Since we dont have any round edges we need to make sure that we give these edges a fillet.

Using the fillet tool select the outer edges of the ring and make sure that they are filleted to 1mm

6. Fillet inner Edges

Using the fillet tool select the inner corner edges within the ring. This might take a little bit of maneuvering bit once you are at an angle were you are able to select all the necessary corners.

Once you have selected all the inner edges make sure that the fillets are made to be 0.5mm.

Here is what is should look like.

7. Text

Start off by unselecting your "Selective Analysis" on the left side of your screen.

We are now going to be using yet another. We are going to be using the tool "Tangent Plane". Make your way to the create drop down menu and select tangent line. From there select the side wall of the ring.

It should look something like this

Right click the plane and create a new sketch on that plane.

When you create a new sketch make sure that you follow closely to the tutorial and make sure that the measurements are the same 80.4L and 8mmH

You can add another new tool to the list and add some text. Select the text box tool and follow the following steps.

In the text box settings select your font, write your name, select the height to 4mm, character spacing to 20mm and make sure that it is properly alligned.

Again, we are going to be using another new tool. the Emboss tool. Select your text and the side wall of the ring.

It should look something like this but we need the text to be set in the ring. Go back to your settings on the right and select the second effect with a depth of -0.33mm or else it will not work.

It should look like this.

Here is the final product.
